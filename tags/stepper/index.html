Indicateur d’étapes (stepper)

Voir la page de documentation du composant sur le Système de Design de l’État Ouvre une nouvelle fenêtre

Documentation du tag

Returns a stepper item. Takes a dict as parameter, with the following structure:

data_dict = {
    "current_step_id": "Number of current step",
    "current_step_title": "Title of current step",
    "next_step_title": "(Optional) Title of next step",
    "total_steps": "Total number of steps",
}

All of the keys of the dict can be passed directly as named parameters of the tag.

Tag name: dsfr_stepper

Usage: {% dsfr_stepper data_dict %}

Exemples

Données 

{'current_step_id': '1',
 'current_step_title': 'Titre de l’étape en cours',
 'next_step_title': 'Titre de la prochaine étape',
 'total_steps': '3'}

Résultat

Étape 1 sur 3 Titre de l’étape en cours

Étape suivante : Titre de la prochaine étape

Données 

{'current_step_id': '4',
 'current_step_title': 'Titre de la dernière étape',
 'total_steps': '4'}

Résultat

Étape 4 sur 4 Titre de la dernière étape
